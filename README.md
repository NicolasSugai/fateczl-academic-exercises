# FATEC ZL - Academic Exercises

This repository contains all my exercises, lab guides, and code developed during my Analysis and Systems Development (ADS) degree at FATEC Zona Leste.

## 📁 How it is organized:
* **By Semester:** Folders like `1st-semester`, `2nd-semester`.
* **By Subject:** Inside each semester, folders are named after the class (e.g., `LP` for Logic Programming).
* **By File:** Standard naming like `L01E04_recursiveness` to keep everything clean and easy to find.

The main goal here is to practice Git, track my coding evolution, and showcase my academic progress.
